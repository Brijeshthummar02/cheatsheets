# Flaticon Icon Placeholders

Drop your licensed Flaticon exports in this folder using the exact names below:

- java.png
- springboot.png
- dsa.png
- git.png

The redesigned UI is already wired to these paths:

- /icons/flaticon/java.png
- /icons/flaticon/springboot.png
- /icons/flaticon/dsa.png
- /icons/flaticon/git.png

If any file is missing, the app automatically falls back to built-in vector icons.

Suggested export settings from Flaticon:

- PNG or SVG format
- Transparent background
- Recommended size: 128x128 or 256x256
- Consistent line thickness across all 4 icons

Attribution and licensing must follow your selected Flaticon license terms.
